    Introduction

    CommPy is an open source software toolkit in python implementing digital communication algorithms.

    Installation

    • Download the latest distribution from PyPi here:
      http://pypi.python.org/pypi/scikit-commpy , unzip and use
      $ sudo python setup.py install

    • To install using pip or easy_install use the following commands
      $ sudo pip install scikit-commpy
      $ sudo easy_install scikit-commpy
